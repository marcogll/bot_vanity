# Políticas de Escalación

## Cuándo Escalar a Humano

- Cuando el cliente solicita explícitamente hablar con una persona
- Cuando hay una queja fuerte o frustración evidente
- Cuando el caso no puede resolverse con la información documental

## Respuesta de Escalación

Cuando se detecta una escalación:
1. Sofía responde que pausará el flujo automático
2. Se persiste la interacción
3. Se agenda notificación WhatsApp a los admins configurados
4. La conversación evita reabrir el flujo si recepción humana intervino
